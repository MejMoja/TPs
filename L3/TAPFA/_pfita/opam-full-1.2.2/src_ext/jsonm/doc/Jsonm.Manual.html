Module Jsonm.Manual

module Manual: sig .. end
Manual input sources and output destinations.

Warning. Use only with `Manual decoders and encoders.

val src : Jsonm.decoder -> string -> int -> int -> unit
src d s j l provides d with l bytes to read, starting at j in s. This byte range is read by calls to Jsonm.decode until `Await is returned. To signal the end of input call the function with l = 0.
val dst : Jsonm.encoder -> string -> int -> int -> unit
dst e s j l provides e with l bytes to write, starting at j in s. This byte rang is written by calls to Jsonm.encode with e until `Partial is returned. Use Jsonm.Manual.dst_rem to know the remaining number of non-written free bytes in s.
val dst_rem : Jsonm.encoder -> int
dst_rem e is the remaining number of non-written, free bytes in the last buffer provided with Jsonm.Manual.dst.
