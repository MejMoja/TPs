  • le fichier Doxyfile fourni (que vous ne devez pas modifier).
Makefile
Utiliser l'utilitaire make pour effectuer les traitements suivants
  • Compiler le programme 
    $ make [all]
  • Tester le programme avec valgrind 
    $ make test
  • Générer cette documentation HTML avec doxygen 
    $ make doc
  • Supprimer les fichiers objets et exécutables (nettoyer) 
    $ make clean
  • Supprimer les fichiers objets et exécutables ainsi que cette documentation (effacer) 
    $ make clear
Voir également
Une documentation des entêtes et fonctions C standards est disponible dans le manuel Unix, section 3.
  • Pour l'entête string.h : 
    $ man -s3 string
  • Pour la fonction printf() : 
    $ man -s3 printf
    Comparer le résultat de la commande précédente avec celui des commandes suivantes : 
    $ man printf 
    $ man -s1 printf
