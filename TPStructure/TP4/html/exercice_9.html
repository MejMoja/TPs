Exercice 9

Gérer dynamiquement une collection d'éléments dynamiques

Avertissement
Nécessite d'avoir terminé l'exercice 2.
Collection de taille variable : une représentation possible
En C, les chaînes de caractères sont des tableaux d'éléments de type char, dont la fin est signalée par le caractère nul, i.e. '\0'.
Dans cet exercice, l'objectif est de manipuler des tableaux dynamiques contenant des chaînes de caractères dynamiques en nombre variable.
Une convention de représentation possible est la suivante : manipuler des tableaux d'éléments de type char *, dont la fin est signalée par le pointeur nul, i.e. NULL.
Un tel tableau sera représenté par une variable de type char ** ; l'adresse d'un tel tableau sera stockée dans une variable de type char ***.
